# Promo post — homelab-hybrid-on-purpose (LinkedIn, paste as-is)

I run 27 public services from a Mac mini on my desk — and that's the point.

The lab exists to keep me hands-on. Reading about Iceberg catalogs, MCP servers, or LLM serving is not the same as running them, breaking them, and wiring them into automations that kill mundane chores. Everything I want to understand earns a container first.

The rest is hybrid on purpose: Hetzner, Oracle Cloud's always-free VMs, a small AWS EC2, Fly.io, Frappe Cloud, Vercel — plus Supabase Cloud, MongoDB Atlas, and MotherDuck for data. Tailscale melts all of it into one flat WireGuard mesh with the box on my desk. Zero open router ports; one outbound Cloudflare Tunnel answers everything, and anything unrecognized gets a 404.
